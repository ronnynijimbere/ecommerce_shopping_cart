# ecommerce_shopping_cart
A Nextjs/typescrypt app ecommerce shopping cart

To run this project :
1. cd to project folder (ecommerce_shopping_cart)
2. yarn install
3. yarn run dev
4. The Project runs on http://localhost:3000/
